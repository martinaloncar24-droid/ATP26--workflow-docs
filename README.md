# ATP26 – Workflow dokumentacija

Ovaj repozitorij sadrži dokumentaciju vezanu uz frontend i backend tehnologije,
alate te proces deploymenta. Dokumenti su izvorno napisani u Markdown formatu
te su konvertirani u AsciiDoc i HTML pomoću alata Pandoc.

## Struktura projekta

- `source/markdown` – izvorne Markdown datoteke
- `asciidoc` – konvertirane AsciiDoc datoteke
- `output_html` – HTML verzije dokumenata
- `images` – slike korištene u dokumentima

## Dokumenti

- Introduction
- Frontend Technologies
- Backend Technologies
- Tools & Deployment

## Alati

- Markdown
- Pandoc
- Visual Studio Code
